Правила форматирования статей

В статье сохраняются картинки, параграфы, а также жирный, курсив и подчеркивание. Остальное форматирование не учитывается. Для форматирования статей, используйте следующие обозначения в тексте:
  • 1> - заголовок 1 уровня (название статьи). Обязательно должен быть один.
  • 2> - заголовок 2 уровня. Их может быть любое количество.
  • 3> - заголовок 3 уровня. Их может быть любое количество.
  • >> - Подпись под картинкой.
  • A> - строка с именем автора статьи.
  • [[многострочный текст]] - Не разбивать на параграфы (например, при форматировании стихов).
  • [R]многострочный текст[/R] - Расположить справа (например, для эпиграфа). Теги [R] и [/R] должны быть каждый на отдельной строке.
